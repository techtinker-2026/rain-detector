🌧️ Arduino Rain Detector Project using LCD & Buzzer 🔔

In this video, I made a simple Rain Detection System using Arduino UNO, rain sensor, buzzer, and 16x2 LCD display. The system detects rain/water drops and gives an alert using a buzzer while displaying the status on the LCD.

🛠️ Components Used:
• Arduino UNO
• Rain Sensor Module
• 16x2 I2C LCD Display
• Buzzer
• Jumper Wires

⚡ Features:
✅ Detects rain/water
✅ Buzzer alert system
✅ LCD status display
✅ Beginner-friendly Arduino project
